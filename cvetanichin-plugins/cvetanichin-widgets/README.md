# Cvetanichin Widgets Plugin

Custom widget set for the Cvetanichin WordPress theme.
Seven widgets, all styled to the Cvetanichin design system.

## Installation
Upload to `/wp-content/plugins/` and activate. Then add widgets via **Appearance → Widgets**.

## Widgets included
- CTA Block — headline + body + button (accent / inverse / plain)
- Bio Card — portrait + name + role + bio
- Stat Row — up to 3 editorial figures (dark/light)
- Newsletter — email opt-in with shortcode support
- Recent Posts — brand-styled post list
- Product Card — price card with accent strip
- Pull Quote — italic serif quote (inset / plain / accent)

See main theme README for full documentation.
